Project - Pegasus
Pegasus is an e-learning web application, which consists of three main modules. These are Coursespace, Workspace and MyPegasus. This project was intended to be developed as a centrally hosted application that provides advanced assessment capabilities as well as basic course management capabilities. Pegasus includes modules like Content Structure Management, Question Bank Management, Assessment Management, Framework Management, Result and Report Management, and Search and Grade Book. This system helps instructors or students have remote interaction through the internet, thus making the jobs easier when creating permissions and settings depending on user rolls. Pegasus has been implemented in a layered approach. Students can take the assessment scheduled by an instructor and get their results.
